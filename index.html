Application

O coração de equipamentos Pedal Pi. Gerencia o pedalboard atual, facilita o processo de armazenamento de dados e permite a expansão do controle através de componentes.

Components

Application sozinho é como uma CPU sem mouse, monitor e teclado. Componentes permitem o acesso às funcionalidades a um usuário através de botões, displays e até mesmo aplicativos.
